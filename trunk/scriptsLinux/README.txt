Dans ce dossier il y a des scripts pour linux pour tracer des graphes à partir des fichiers de log.
Il faut avoir gnuplot d'installé et de quoi dessiner des png (mais ça ça devrait aller)

Pour l'utiliser, mettez les fichiers de log dans ce dossier et lancer le script ./parseAll.sh
Il faudra surement faire chmod +x *.sh dans le cas où ils ne sont pas créés comme exécutable par svn.

Le résultat est dans le dossier out qui sera créé au lancement du script.

enjoy.
